Serotype Ranking by Lowest Ct Value (Ct < 40 only) v0.4

Upload a CSV file. The first column is ignored for ranking (e.g., sample ID / piaB). All subsequent columns are serotype assays with Ct values.
For each row, only assays with Ct < 40 are considered. The lowest Ct gets rank 1, second lowest rank 2, etc. (up to rank 5).
If an assay has Ct ≥ 40 or is missing, it does not appear in the ranking at all. If no assay has Ct < 40, all rank entries are "-" (name) and NA (Cq).
