Serotype Ranking by Lowest Ct Value v0.2

Upload a CSV file. The first column is ignored (e.g., sample ID). All subsequent columns are serotype assays with Ct values.
For each row, the assay with the lowest Ct gets rank 1, second lowest rank 2, etc.
If Ct ≥ 40 or missing, the assay name is replaced by "-". The ranking uses only numeric values < 40.
